# 🚀 ApplySaathi

ApplySaathi is a Flask-based web application designed to help users apply for forms, manage applications, and handle administrative workflows efficiently.

Built with:
- 🐍 Flask (Backend)
- 🐘 PostgreSQL (Supabase)
- 🔐 Flask-Login Authentication
- 🗄 SQLAlchemy ORM
- 🚀 Hosted on Render

---

👉 https://your-render-url.onrender.com

## 📌 Features

- 👤 User Registration & Login
- 🔐 Secure Authentication (Flask-Login)
- 👨‍💼 Admin & Superadmin Roles
- 🗂 Form/Application Management
- 🧾 Database Integration with PostgreSQL
- 🔑 Environment-based Configuration
- 🚀 Production Deployment with Gunicorn
